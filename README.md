# Hong Kong Tunnel Fees Web App for Private Car (WHC, CHT, EHC, TLT)

A simple, mobile-friendly web application to check real-time and future toll fees for Hong Kong's major tunnels (Western Harbour Crossing, Cross Harbour Tunnel, Eastern Harbour Crossing, and Tai Lam Tunnel) for private car.

## Features

*   **Real-time Pricing:** Shows current toll fees based on the time of day.
*   **Time-Varying Tolls:** accurately calculates "Time-varying Tolls" including peak, normal, off-peak, and transition periods (every 2 minutes).
*   **Timeline View:** Displays a schedule of upcoming price changes.
*   **Holiday Support:** Toggle for Sunday/Public Holiday pricing.
*   **Mobile Optimized:** Sticky footer controls and responsive layout.
